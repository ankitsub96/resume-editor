# Resume Editor

An interactive, browser-based resume editor built with React + Vite.

## Features

- Two-column drag-and-drop layout with resizable divider
- Drag to reorder sections, items, and bullet points
- Live editable text fields — all changes auto-saved to localStorage
- Theme switcher, font controls, spacing controls
- PDF export with SEO keywords embedded as metadata
- GitHub Pages deployment

## Live Demo

https://ankitsub96.github.io/resume-editor/

## Local Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

Builds the app and pushes to the `gh-pages` branch automatically.
